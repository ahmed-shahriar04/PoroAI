About & Documentation

Poro Mentor is an AI study workspace for students.

Poro Mentor helps students understand topics, summarize PDFs and images, find learning resources, build resumes, review career plans, solve coding problems, and organize productivity in one simple student-friendly platform.

What it does

Main Features

These are the core modules included in Poro Mentor.

Study Helper

Explains difficult topics with examples, key points, common confusion, practice questions, and flashcards.

PDF / Notes Summarizer

Supports PDF and image attachments. Students can upload files, write instructions, then get summaries and study notes.

Learning Resources

Suggests valid links for videos, documents, websites, books, documentation, and learning platforms.

Resume Builder

Step-by-step resume/CV builder with templates, profile photo, live preview, AI review, and PDF export.

Coding Helper

Explains code concepts, examples, common mistakes, practice tasks, and useful documentation links.

Career Guide

Creates roadmaps, skill plans, project ideas, interview preparation, and career learning paths.

Productivity Planner

Builds study routines, exam plans, weekly goals, focus strategies, and revision schedules.

Campus Assistant

Helps with student life, academic planning, campus tasks, course decisions, and club activities.

How to use

User Documentation

Simple guide for using the main parts of the website.

01

Open Poro Mentor

Go to the Mentor page and choose an agent mode from the left sidebar, such as Study Helper, PDF / Notes, Coding Helper, or Career Guide.

02

Ask with text, PDF, or image

Type your question normally. For files, click Attach, select one or multiple PDFs/images, write your instruction, then press send.

03

Use response tools

You can copy AI replies, retry answers, edit your previous prompt and resend it, open recommended links, or export chat history.

04

Build a resume

Open Resume Builder, choose a template, add personal information, photo, education, skills, projects, and experience, then export as PDF.
