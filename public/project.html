Poro Mentor

An AI-powered student workspace that combines general chat, study help, PDF/image understanding, resume building, coding help, career guidance, productivity planning, learning resources, and image generation.

Try Poro Mentor Open Resume Builder
Built with OpenAI Student-first AI assistant

Problem

Students often need different tools for studying, summarizing notes, reviewing resumes, finding resources, planning careers, solving coding problems, and creating visual assets. Switching between many apps wastes time and creates friction.

Solution

Poro Mentor brings these workflows into one clean AI workspace. Students can chat naturally, attach PDFs/images, generate study notes, create resumes, review career plans, get coding help, and generate images from prompts.

Target Users

Students from school, college, and university who need a simple AI companion for learning, productivity, career preparation, and academic support.

Core Features

What Poro Mentor can do

Default Chat with general GPT-style assistant behavior
Study Helper with explanations, practice questions, and flashcards
Multiple PDF and image attachment analysis with user instructions
Resume Builder with templates, live preview, AI review, and PDF export
Coding Helper for concepts, examples, bugs, and documentation links
Career Guide with roadmaps, skills, projects, and learning plans
Productivity Planner for study routines and exam preparation
AI image generation directly inside the mentor chat box
Valid resource links for videos, books, documents, websites, docs, and courses
Local chat history, single chat delete, edit prompt, retry response, and copy answer
OpenAI Usage

AI Workflows

  • Responses API for chat and agent answers
  • GPT model dropdown loaded from API-supported models
  • Multimodal understanding for uploaded images
  • PDF summarization after text extraction
  • Resume review and improvement suggestions
  • Image generation using prompt-based image endpoint
Built With

Tech Stack

  • LocalStorage chat history

API Endpoints

Backend endpoint map

These are the main backend routes used by the project.

GET/api/healthChecks server health.
GET/api/modelsLoads GPT models available to the API key.
POST/api/agentMain AI chat and agent endpoint.
POST/api/agent-filesHandles multiple PDFs/images with user prompt.
POST/api/pdfSingle PDF summarization endpoint.
POST/api/imageSingle image understanding endpoint.
POST/api/generate-imageGenerates images from user prompts.
